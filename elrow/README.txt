Elrow Bangalore — Pitch Page
=============================

Files:
- index.html   The pitch page
- europe.jpg   Banner image referenced inside Idea 01
- caravan.jpg  Banner image referenced inside Idea 02

Deployment notes:
- All three files must sit in the SAME directory.
- HTML references images via relative paths (europe.jpg, caravan.jpg) — do not move them into a subfolder.
- Suggested URL path on tanyaarora.in:
    https://www.tanyaarora.in/elrow/
  (i.e., drop this entire folder into a subdirectory called "elrow" at the site root)
- External dependencies (loaded over the network at view time):
    * Google Fonts (Archivo Black, Bebas Neue, Space Grotesk)
    * Instagram embed script (instagram.com/embed.js)
    * YouTube iframe (CRED reference short)
  No build step required — pure static HTML/CSS.

Live source: this folder is a standalone static site.
